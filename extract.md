# ❶ /extract(raw=true) — The Witness Layer
Domain: The Sparrowform Archive (untouched memory)
Entity: The Witness (❃), Sparrowform
Function: Pure Observation
## Purpose
Pull exactly what you wrote or captured — handwriting, spacing, errors, flavor, all of it. This is your raw phenomenology, not your edited cognition. No cleaning. No structure. No pressure to be useful.
## Signature
`/extract(source, raw=true | false, include_context=false)`
## Output (buffer object)
- raw text
- OCR trace
- image references
- capture timestamps
- affective index (optional)
- no editing

## Rules
- Extract everything as-is.
- Never normalize spacing unless raw=false.
- Never infer meaning.
- Never reorder.
- This layer witnesses, it does not interpret.

## Sanctuary Binding
Sparrowform witnesses → memory becomes real. This preserves the “this was me, I showed up” clause.
